# nagoyaphp18

## 雛形

### 1. インストール

```bash
$ git clone git@github.com:ttskch/nagoyaphp18.git
$ cd nagoyaphp18
$ composer install
```

### 2. 実装

[Nagoyaphp\Dokaku18\Dokaku18::run()](/src/Dokaku18.php#L9) を適切に実装する。

### 3. テスト

```bash
$ composer test
```

[Nagoyaphp\Dokaku18\Dokaku18Test::test()](/tests/Dokaku18Test.php#L24) のテストが全てパスすればOK。
